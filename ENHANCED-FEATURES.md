Beyond categories, the system identifies **themes**:

- **warning** - Cautionary messages

### 3. **Importance Scoring** ⭐
Every event gets an importance score (0-20):
- **⭐⭐⭐ High Significance** (15+): Major predictions, world events
- **⭐⭐ Significant** (10-14): Important spiritual messages
- **⭐ Notable** (5-9): Meaningful events
- Events are visually highlighted based on importance!

### 4. **Sentiment Analysis**
Events are analyzed for sentiment:
- ✨ **Positive** - Uplifting, peaceful messages
- ⚠️ **Concerning** - Warnings, important alerts
- 📝 **Neutral** - Informative content

---

### 5. **Smart URL Handling**
URLs are automatically:
- **Extracted** from text
- **Categorized** by platform (Facebook, Instagram, YouTube, etc.)
- **Displayed beautifully** with clickable links
- **Labeled** with type (Facebook Post, YouTube Video, etc.)

### 6. **Storyline Connections** 📖
The parser now connects related events:
